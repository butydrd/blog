Windows 重命名网络适配器以及切换网络类型

#切换网络类型

Windows 防火墙可以针对专用网络和公用网络配置，但是切换只能切换主网络，像虚拟网络这些就没法通过设置切换 先获取当前网络连接详情

Get-NetConnectionProfile

找到需要更改一项的 InterfaceAlias 或者 InterfaceIndex，输入以下命令更改（需要管理员运行）

Set-NetConnectionProfile -InterfaceAlias {alias} -NetworkCategory "Private"

# or
# Set-NetConnectionProfile -InterfaceIndex {index} -NetworkCategory "Private"

#更改网络名称

需要通过注册表更改，按 Win + R 输入 regedit 进入注册表，输入以下路径 HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\NetworkList\Profiles

在 Profiles 目录查看每一个 UUID 条目 找到对应的网络，然后修改 ProfileName 字段为想要的名称，此时进入控制面版在 控制面板\网络和 Internet\网络和共享中心 就应该变成想要的名称了，在旁边的 更改适配器设置 里面可以更改对应的网络适配器名称，最终效果如下
